Linux下maven安装

Thu, Mar 21, 2019

1. 下载

下载地址http://maven.apache.org/download.cgi

$wget -c http://mirrors.hust.edu.cn/apache/maven/maven-3/3.6.0/binaries/apache-maven-3.6.0-bin.tar.gz

2. 安装

解压到安装目录/opt

#tar -xvf apache-maven-3.6.0-bin.tar.gz -C /opt

3. 配置环境变量

#vi /etc/profile
export MAVEN_HOME=/opt/apache-maven-3.6.0
export PATH=$PATH:$MAVEN_HOME/bin

4. 配置仓库路径

配置依赖包存储路径：

#vi /opt/apache-maven-3.6.0/conf/settings.xml
<localRepository>/opt/repo-maven-3.6.0/</localRepository>

配置本地maven私服：

#vi /opt/apache-maven-3.6.0/conf/settings.xml
    <!-- 指定maven私服 -->
    <profile> 
    <id>mynexus</id>  
    <repositories> 
        <repository> 
            <id>nexus_public</id>  
            <url>http://172.31.134.221:8081/nexus/content/groups/public/</url>  
            <releases> 
                <enabled>true</enabled> 
            </releases>  
            <snapshots> 
                <enabled>true</enabled> 
            </snapshots> 
        </repository> 
    </repositories>  
    <pluginRepositories> 
        <pluginRepository> 
            <id>nexus_public</id>  
            <url>http://172.31.134.221:8081/nexus/content/groups/public/</url>  
            <releases> 
                <enabled>true</enabled> 
            </releases>  
            <snapshots> 
                <enabled>true</enabled> 
            </snapshots> 
        </pluginRepository> 
    </pluginRepositories> 
    </profile>

    <activeProfiles>
        <activeProfile>mynexus</activeProfile> 
    </activeProfiles>

4. 测试

#source /etc/profile
#mvn -version

5. 用法

5.1 一键编译

在java代码根目录下执行：

#mvn package

会自动下载依赖包，并对源码进行编译打包
